Recipe Box

Recipe Box is an app for selecting recipes and automatically combining ingredients into one shopping list.

Recipe Box utilizes a recipe database API and a RESTful API that parses ingredients lists so they can be collated onto a grocery list. This is a front-end application and was built with:

  • HTML
  • CSS & CSS FlexBox
  • JavaScript
  • AJAX
  • 2 APIs
  • PostgreSQL

My primary responsibility was building a responsive page layout, ensuring a great user experience across all devices.

Habit City

Habit City is a habit tracking app that helps users to build healthier routines by logging their progress.

Habit City utilizes 3 databases, one for user information, one for users entered habits and finally one for comments. CRUD operations were used for adding, editing and deleting input on the Habits Board and Manage Habits Board. This is a full-stack application and was built with:

  • HTML
  • CSS
  • JavaScript
  • Node JS
  • Express
  • PostgreSQL

My responsibilities included building a mobile-responsive list page layout and routing information between the app and a PostgreSQL database.
